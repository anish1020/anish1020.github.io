## 🏗️ Architecture & Key Features

This portfolio is built with a focus on high performance, accessibility, and modern UI/UX design patterns, utilizing raw web technologies without heavy frontend frameworks to ensure rapid load times.

- **Performant UI/UX**: Custom-built using semantic HTML5, CSS3 with advanced variables for seamless theming (Dark/Light mode), and Vanilla JavaScript for interactivity.
- **Progressive Web App (PWA)**: Fully installable as a standalone application. Utilizes dynamic Service Workers (`sw.js`) and a Web App Manifest for offline caching and native-like mobile experiences.
- **Interactive Micro-Animations**: Employs the `IntersectionObserver` API for highly optimized scroll-triggered animations and custom JavaScript for dynamic typewriter effects, avoiding heavy third-party animation libraries.
- **Search Engine Optimization (SEO)**: 
  - Comprehensive injection of `JSON-LD` structured data schema mapping for people and organizations.
  - Carefully mapped Open Graph (OG) and Twitter Card meta tags for rich social media embedding.
  - Strictly defined Canonical URLs, dynamic `sitemap.xml`, and optimized `robots.txt` routing.
- **Recruiter Matrix View**: Features a unique UI toggle mechanism that seamlessly transforms the graphical portfolio into a high-density, text-heavy, ATS-friendly matrix view catered specifically to technical recruiters.
- **Secure Contact Architecture**: Form submissions are securely routed through a serverless Google Apps Script endpoint, leveraging CORS and Fetch API structures to process inbound contact requests without exposing backend mechanics.

## 💻 Local Development Setup

To run this repository locally for development or testing overrides:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/anish1020/anish1020.github.io.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd anish1020.github.io
   ```
3. **Start a local development server:**
   You can use tools like Live Server (VS Code Extension), Python's HTTP server, or Node's `serve` package.
   
   *Using Python:*
   ```bash
   python -m http.server 5500
   ```
   *Using Node/npx:*
   ```bash
   npx serve .
   ```
4. **Access the site:**
   Open your browser and navigate to `http://localhost:5500`

> **Note:** Client-side source protection features (disabling context menus, specific keyboard shortcuts) are dynamically disabled when serving the site from `localhost` or `127.0.0.1` to ensure frictionless development.
