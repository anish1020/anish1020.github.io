## 🚀 Key Features

- **Modern & Responsive UI**: Designed with raw HTML, CSS, and vanilla JS. Built with pure custom styling and CSS variables for theming.
- **PWA Ready**: Installable Progressive Web App with offline support via service workers.
- **Micro-Animations**: Uses modern CSS animations, Intersection Observers, and custom JS text typing effects.
- **SEO Optimized**: Fully equipped with semantic HTML, JSON-LD structured data, metadata, Open Graph (OG), Twitter Cards, XML sitemap, and `robots.txt`. Built and optimized for web crawlers.
- **Recruiter View**: Special UI toggle that converts the visual portfolio into a dense, resume-style single-page format for technical recruiters and ATS parsing.
- **Security**: Basic client-side protections (text selection, copy restrictions, DevTools disabling) enabled in production for casual protection, skipped cleanly in `localhost` development.
- **Contact Form**: Connected securely to Google Apps Script endpoints to ensure reliable message delivery without exposing API keys.
